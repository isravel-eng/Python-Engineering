# Python Engineering

A structured repository focused on building strong Python engineering foundations for backend development, machine learning engineering, automation, APIs, data processing, and production-grade software development.

This repository is designed as a long-term engineering knowledge base covering:

- Core Python
- Object-Oriented Programming
- Exception Handling
- Iterators and Generators
- Decorators
- Context Managers
- File Handling
- Standard Library
- APIs
- Databases
- NumPy
- pandas
- Engineering Mini Projects
